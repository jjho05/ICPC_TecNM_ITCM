# ICPC TecNM - Instituto Tecnológico de Ciudad Madero

### 🚀 Plataforma de Concursos de Programación
Sistema integral desarrollado para la gestión de concursos estilo ICPC dentro del TecNM Campus ITCM.

## ✨ Características
- **Backend Realtime**: Integración completa con **Supabase** para persistencia y actualizaciones en vivo.
- **Roles de Usuario**: Soporte para Administradores, Jueces (Profesores), Coaches y Alumnos.
- **Arena de Competencia**: Interfaz optimizada para resolución de problemas y seguimiento de veredictos.
- **Scoreboard Dinámico**: Clasificación en tiempo real basada en reglas estándares del ICPC.

## 🛠 Tecnologías
- **Frontend**: HTML5, CSS3 Moderno, Javascript Vanilla (ES6+).
- **Base de Datos**: PostgreSQL vía Supabase.
- **Hosting**: Optimizado para Render.

## ⚙️ Instalación Local
Para ejecutar en local, se requiere un servidor web (debido al uso de ES Modules):
```bash
npx http-server -p 8080
```
